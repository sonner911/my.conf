# Manual configuration

## ~/.zshrc
Add `source "$HOME/.my.conf/zshrc"`

Set theme to `mine`

## ~/.profile
Include `source ~/.my.conf/bash_profile`

## ~/.bash_profile
Add `source ~/.profile`

## ~/.shellrc
Put common initialization for Bash and Z shell

## Customize Oh my Zsh
`ln -s ~/.my.conf/zsh_custom ~/.oh-my-zsh/custom`
